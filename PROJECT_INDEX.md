## üìä 10 Playwright Tools Available

```
1. launch_browser
   Purpose: Start browser instance
   Parameters: browserType (chromium|firefox|webkit), headless, url
   Returns: browser-N ID

2. navigate
   Purpose: Go to URL
   Parameters: browserId, url, waitUntil (load|domcontentloaded|networkidle)
   Returns: navigation status

3. click_element
   Purpose: Click page element
   Parameters: browserId, selector (CSS)
   Returns: click status

4. fill_input
   Purpose: Type into input field
   Parameters: browserId, selector, text
   Returns: fill status

5. get_text
   Purpose: Extract text content
   Parameters: browserId, selector
   Returns: text content

6. get_title
   Purpose: Get page title
   Parameters: browserId
   Returns: page title

7. get_url
   Purpose: Get current URL
   Parameters: browserId
   Returns: current URL

8. screenshot
   Purpose: Take page screenshot
   Parameters: browserId, filename (optional)
   Returns: file path

9. wait_for_navigation
   Purpose: Wait for page navigation
   Parameters: browserId, timeout (ms)
   Returns: completion status

10. close_browser
    Purpose: Close browser instance
    Parameters: browserId
    Returns: close status
```
